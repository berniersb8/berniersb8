## 🔮 What I'm Working On

I'm currently building **[Arcanic](https://github.com/arcanic-kit)** — a modular open-source toolkit for .NET developers who care about Clean Architecture.

> *Build cleaner, scalable, and maintainable applications with ease.*

Arcanic provides lightweight, zero-dependency libraries that implement modern architectural patterns without the boilerplate overhead:

| Package | Status | Description |
|---|---|---|
| [`Arcanic.Mediator`](https://github.com/arcanic-kit/Mediator) | ✅ Available | Mediator pattern with CQRS — decouple your in-process messaging |
| [`Arcanic.Result`](https://github.com/arcanic-kit/Result) | ✅ Available | Functional Result pattern — replace exceptions with explicit error handling |
| `Arcanic.Repository` | 🚧 Coming Soon | Generic repository abstractions for clean data access |
| `Arcanic.MinimalApi` | 🚧 Coming Soon | Structured, testable Minimal API extensions |
